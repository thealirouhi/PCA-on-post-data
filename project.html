To prevent the model from being biased, we apply the Normalization method to the data.

In this way, for each pixel (feature), we find the minimum and maximum data. Then, we subtract each data from the minimum, divide it by the range of its feature, multiply by 2, and subtract one from it.

With this method, the data for each pixel ranges between -1 and +1.

QR Algorithm!

To use this algorithm to obtain eigenvalues and eigenvectors, we define two functions.

qr_decomposition

This function takes an n by n backmatrix as input and outputs the matrices Q and R, which are its decompositions.

qr_algorithm

This function decomposes the matrix Ak at each step using the previous function and produces the matrix Ak+1.

Given that as n increases in the matrix An, our accuracy in obtaining eigenvectors and eigenvalues increases, here we have limited ourselves to the matrix A100.

In [21]:
def qr_decomposition(A):
  n = A.shape[0]
  Q = np.zeros((n, n), dtype=np.float64)
  R = np.zeros((n, n), dtype=np.float64)

  for i in range(n):
    v = A[:, i]
    for j in range(i):
      R[j, i] = np.dot(Q[:, j], A[:, i])
      v = v - R[j, i] * Q[:, j]

    R[i, i] = np.linalg.norm(v)
    Q[:, i] = v / R[i, i] if R[i, i] != 0 else 0

  return Q, R

def qr_algorithm(A):
  n = A.shape[0]
  A2 = A.copy()
  V = np.eye(n)

  for _ in range(100):
    Q, R = qr_decomposition(A2)
    A2 = R @ Q
    V = V @ Q

  eigen_values = np.diag(A2)
  eigen_vectors = V

  return eigen_values, eigen_vectors

Here, using the previous algorithm and the functions we wrote, we extract the eigenvalues and eigenvectors from the covariance matrix.

Given that the eigenvectors have the largest eigenvalues in order, we separate the first 3 eigenvectors from all the eigenvectors.

To obtain the coordinates of each data sample based on the eigenvectors we obtained, we need to map each data sample onto its eigenvector, which is done by dot-multiplying the eigenvector by the data sample vector.
